## 🏗️ 系统架构

### 技术栈

- **后端语言**: Go 1.25.3
- **Web框架**: Gin
- **ORM**: GORM
- **数据库**: MySQL 8.0 / SQLite3
- **缓存**: Redis (可选)
- **认证**: JWT
- **加密**: bcrypt, SHA256
- **前端**: HTML5 + CSS3 + JavaScript
- **容器化**: Docker + Docker Compose
