## 注册与登录

### 新用户注册

1. 访问网站首页，点击右上角**注册**按钮
2. 填写注册信息：
   - **用户名**: 3-50个字符，支持字母、数字、下划线
   - **邮箱**: 用于接收通知和找回密码
   - **密码**: 至少6个字符
   - **确认密码**: 再次输入密码
   - **邀请码**: 可选，有邀请码可获得奖励
3. 点击**注册**完成注册
4. 系统会自动发送欢迎邮件（如果邮箱验证已启用）

### 用户登录

1. 点击**登录**按钮
2. 输入用户名或邮箱及密码
3. 可选择**记住我**保持登录状态
4. 点击**登录**进入系统

### 忘记密码

1. 在登录页面点击**忘记密码**
2. 输入注册邮箱
3. 查看邮箱中的重置链接
4. 按照提示重置密码

## 浏览资源

### 首页浏览

- **最新资源**: 显示最新上传的资源
- **热门资源**: 显示下载量最高的资源
- **推荐分类**: 点击分类名称查看该分类下的所有资源

### 分类浏览

1. 点击顶部导航栏的**分类**
2. 选择子分类
3. 浏览该分类下的所有资源
4. 可按时间、下载量、评分排序

### 资源详情

点击任意资源卡片进入详情页，查看：
- 资源标题和描述
- 资源内容预览
- 分类信息
- 上传者信息
- 积分价格
- 下载量
- 评分和评论

## 搜索资源

### 简单搜索

1. 在顶部搜索框输入关键词
2. 按回车或点击搜索按钮
3. 查看搜索结果

### 高级搜索

1. 点击搜索框旁的**高级搜索**
2. 设置搜索条件：
   - 关键词
   - 分类
   - 价格范围
   - 排序方式
3. 点击**搜索**

### 搜索技巧

- 使用引号搜索完整短语：`"软件工具"`
- 使用减号排除关键词：`软件 -游戏`
- 使用通配符：`数据库*`

## 下载资源

### 免费资源

1. 进入资源详情页
2. 点击**免费下载**按钮
3. 下载自动开始

### 积分资源

1. 进入资源详情页
2. 查看积分价格
3. 确保积分余额充足
4. 点击**积分下载**
5. 确认扣除积分并下载

### 下载限制

- 未登录用户：每日最多下载3个资源
- 普通用户：每日最多下载10个资源
- VIP用户：无限制（需管理员设置）

## 积分系统

### 积分获取

1. **注册奖励**: 新用户注册获得10积分
2. **邀请奖励**: 每邀请一位好友注册，获得5积分
3. **上传奖励**: 上传的资源被下载可获得积分（可设置）
4. **签到奖励**: 每日签到获得1积分
5. **评论奖励**: 发布有价值的评论获得2积分

### 积分消费

1. **下载付费资源**: 按资源设定的积分扣除
2. **商城兑换**: 使用积分兑换实物或虚拟物品
3. **打赏作者**: 将积分打赏给资源上传者

### 积分记录

1. 登录个人中心
2. 点击**我的积分**
3. 查看积分余额和获取/消费记录

### 积分规则

- 积分有效期：永久有效
- 不可转账或提现
- 违规行为可能导致积分清零
- 管理员可调整积分规则

## 邀请好友

### 获取邀请码

1. 登录个人中心
2. 进入**邀请好友**页面
3. 系统自动生成您的专属邀请码和邀请链接

### 邀请方式

1. **分享邀请链接**: 直接发送邀请链接给好友
2. **分享邀请码**: 将邀请码告诉好友，注册时填写
3. **社交分享**: 一键分享到微信、QQ、微博等

### 邀请奖励

1. **邀请人奖励**: 好友注册后，邀请人获得5积分
2. **被邀请人奖励**: 新用户使用邀请码注册获得10积分
3. **多级奖励**: 可设置多级邀请奖励机制

### 邀请统计

在**邀请中心**查看：
- 成功邀请人数
- 累计获得积分
- 邀请排行榜
- 最近邀请记录

## 发表评论

### 发表评论

1. 进入资源详情页
2. 滚动到评论区
3. 填写评论内容和评分（1-5星）
4. 点击**发表评论**

### 评论规则

- 评论内容需客观真实
- 禁止发布广告、垃圾信息
- 禁止恶意评分或刷评
- 评论需要审核后才能显示

### 编辑评论

1. 在**我的评论**中找到该评论
2. 点击**编辑**
3. 修改内容后保存

### 举报评论

发现不当评论可点击**举报**，管理员会及时处理。

## 常见问题

### Q1: 为什么我的资源上传审核不通过？
A: 原因可能包括：
- 资源描述不清晰
- 资源格式不符合要求
- 资源内容违规
- 侵犯版权

### Q2: 积分不够怎么办？
A: 可以通过以下方式获得积分：
- 邀请好友注册
- 每日签到
- 上传优质资源
- 发布有价值的评论

### Q3: 下载失败怎么办？
A: 尝试以下解决方案：
- 检查网络连接
- 更换浏览器
- 清除浏览器缓存
- 联系管理员

### Q4: 如何成为管理员？
A: 管理员由系统创建，普通用户无法主动申请。如表现优秀可能被邀请担任版主。

### Q5: 忘记用户名怎么办？
A: 可以使用邮箱登录，或联系管理员帮助找回。

### Q6: 资源链接失效怎么办？
A: 点击**举报失效**，管理员会及时处理并联系上传者修复。

### Q7: 如何提高积分？
A: 通过以下方式：
- 邀请更多好友
- 上传热门资源
- 发布高质量评论
- 参与网站活动

### Q8: 可以删除自己的账户吗？
A: 可以，在个人中心-账户设置中申请注销。注销后数据无法恢复，请谨慎操作。
